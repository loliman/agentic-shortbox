# Agent Setup

Instructions on how to configure, run, and interact with the AI agents locally and effectively.

1. Install dependencies and build the TypeScript action bundle.
2. Provide `OPENAI_API_KEY` through GitHub Actions secrets when running in GitHub.
3. Use issue comments for commands and optionally use `model:*` labels for model selection.
4. Keep repository governance files (`AGENTS.md`, `docs/`, `specs/`, `plans/`) current so Codex can inspect them directly at runtime.
