# Agent Setup

Instructions on how to configure, run, and interact with the AI agents locally and effectively.

1. Install dependencies and build the TypeScript action bundle.
2. Provide `OPENAI_API_KEY` and/or `GEMINI_API_KEY` through GitHub Actions secrets when running in GitHub.
3. Use issue labels for agent/model selection and issue comments for commands.
4. Keep repository governance files (`AGENTS.md`, `README.md`, `docs/`, `specs/`, `plans/`) current so prompts reflect the intended architecture.
